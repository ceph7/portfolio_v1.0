# Portfolio Backend

## Déploiement rapide

1. Configurez les variables d’environnement dans `.env`.
2. Vérifiez que votre base de données est accessible.
3. Exécutez les migrations : `php artisan migrate`.
4. Lancez l’application : `php artisan serve`.

## Endpoint de contact

Le formulaire de contact utilise l’endpoint suivant :
- `POST /api/contact`
